# Features & Platform

## MVP Must-Have Features

1. Chat input & analysis
   - User pastes or types a conversation and clicks "Analyze".
2. Vibe gauge & emotional summary
   - Show an overall vibe gauge (cool → mixed → tense).
   - Short emotional summary.
3. Key insights & flags
   - Bullet-point insights about patterns, tone, and possible red flags.
4. History of analyses per user
   - Logged-in user sees a list of past analyses with title, date, and vibe label.
5. Basic account system
   - Email/password login.
   - Each user can only see their own analyses.

## Nice-to-Have Later

- File upload for chat exports (WhatsApp, Messenger, etc.).
- Multi-conversation comparisons over time.
- Custom red-flag lists and alerts.
- Shareable read-only links for friends/coaches.
- A deeper "coach mode" with more guidance.

## Platform Decision

- Platform: Responsive web app (mobile + desktop) in the browser.
- No native App Store / Play Store app in v1.
